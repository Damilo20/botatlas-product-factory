# Standardization Rules

Always normalize:

Weight → kilograms

Dimensions → millimeters

Speed → meters/second

Payload → kilograms

Battery → Wh

Runtime → hours

Temperature → Celsius

Power → Watts

Currency → USD (store original value separately)
